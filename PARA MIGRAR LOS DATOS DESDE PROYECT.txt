PARA MIGRAR LOS DATOS DESDE PROYECTO LOCAL A UNA BDD POSTGRES

1. Crear un json con los datos del proyecto

python manage.py dumpdata > data.json

importante!!! abrir el archivo y guardar con encoding utf-8 

2. Conectarse a la bdd postgres y migrar 

python manage.py migrate --run-syncdb 

3. Borrar los datos que vienen por defecto en la base postgres

a) entrar a la shell 

python manage.py shell

b) desde la shell ejecutar

	from django.contrib.contenttypes.models import ContentType
	ContentType.objects.all().delete()

c) salir de la shell con exit()


4. Cargar los datos en la base de datos

python manage.py loaddata data.json
